## Phase 1 以降のゲート条件

新パイプラインで同じ `input/spec` から再生成したとき:

1. **映像フレーム数 (nb_frames) が ±1 以内で一致**（＝尺が±1フレーム）
2. 解像度 `1920×1080`, fps `30/1` を維持
3. コーデック `h264+aac` を維持
4. ファイルサイズは ±5% までの変動を許容（エンコーダ設定差）
5. SHA256 の一致は**求めない**（エンコード非決定性のため）。内容の再現は視覚QAで確認。

## 検証コマンド（再計測用）

```bash
cd packages/tech-geopolitics-channel/output
for f in ep001.mp4 ep003.mp4 ep005.mp4 ep006.mp4 ep007.mp4; do
  echo "=== $f ==="
  ffprobe -v error -show_entries format=duration,size,bit_rate:stream=codec_name,width,height,nb_frames,r_frame_rate \
    -of default=noprint_wrappers=1 "$f"
  sha256sum "$f" | cut -c1-16
done
```
