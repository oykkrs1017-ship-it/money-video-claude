## 2. NotebookLM 分析プロンプト集

### シナリオA：封鎖の定量的影響

```
ホルムズ海峡が封鎖された場合の日本への定量的影響を以下の観点で整理してください。

1. 日本の原油輸入量のうちホルムズ経由の割合（%・バレル/日換算）
2. 国家石油備蓄の現在日数と、封鎖が継続した場合に枯渇するまでの期間
3. 原油価格が1バレル100ドル・150ドル・200ドルに上昇した場合の、日本のGDP・貿易収支・電力料金への影響推計
4. LNG（液化天然ガス）についても同様に、ホルムズ依存度と代替調達の可能性を整理

数値は出典とともに示してください。
```

---

### シナリオB：投資家向け銘柄・セクター分析

```
ホルムズ海峡封鎖リスクが高まった局面で、投資家が注目すべき日本株の銘柄・セクターを整理してください。

【受益が期待されるセクター】
- 石油・ガス上流（INPEX等）、総合商社（LNG権益保有）、再エネ、節電・省エネ技術

【ダメージが大きいセクター】
- 航空、輸送、素材（鉄鋼・化学）、電力

各セクターについて、原油価格上昇との連動性・過去の地政学リスク時の株価動向・現在の財務健全性も含めて整理してください。
また、個人投資家が取れる具体的なヘッジ戦略（原油ETF、ゴールド、円安対応）も示してください。
```

### シナリオC：日本の代替ルート・代替エネルギー策

```
ホルムズ海峡封鎖時に日本が取りうる代替手段を、実現可能性と課題ともに整理してください。

【輸送ルートの代替】
- サウジアラビア「東西パイプライン」（ヤンブー港）経由ルート
- UAE「アブダビ原油パイプライン」経由
- アフリカ・ロシア・北米からの緊急調達

【エネルギー源の代替】
- 米国・カナダのシェールオイル・LNG（価格・輸送コスト）
- オーストラリア LNG（イクシスPJ等）
- 原子力発電所の再稼働加速の可否
- 石炭火力の一時的増強
- 需要側：節電・燃料転換・IEA緊急対応メカニズム

それぞれについて、調達コスト増分・準備期間・政治的障壁を示してください。
```

```

4. イラク戦争（2003年）

各危機について：
- 引き金となった地政学イベント

- 日本経済・株式市場への影響（インフレ率、GDP成長率、日経平均）

- 今回のシナリオとの類似点・相違点

特に「現代の日本は1973年当時より脆弱か、強靭か」という観点で結論を示してください。
```

## 3. YouTube台本生成プロンプト（統合版）

NotebookLMにソースを読み込んだ後、以下をそのまま貼り付けてください。

```
あなたはテクノロジー投資×地政学を解説するYouTubeチャンネルの台本ライターです。
格納されたソースをもとに、以下の構成で台本素材を生成してください。

【動画設定】
- テーマ：ホルムズ危機と日本のエネルギー安保
- 尺：約10分（台本文字数：3,000〜3,500字）
- キャラクター：ずんだもん（解説担当、明るく好奇心旺盛）、四国めたん（ツッコミ・投資家視点）
- 視聴者：20〜50代、投資初心者〜中級者、地政学に関心あり

【台本構成】
1. オープニングフック（30秒）
   - 「もし明日、原油が2倍になったら？」など視聴者を引き込む問いかけ

2. 現状説明（2分）
   - 日本の原油輸入のXX%がホルムズ経由という事実
   - 最新のイラン情勢（数値・具体的事件）

3. 封鎖シナリオの影響（3分）
   - ガソリン・電気代・物価への影響（具体的数値）
   - 株式市場・為替への影響
   - 日本の備蓄日数と「タイムリミット」

4. 日本の対策と課題（2分）
   - 国家備蓄・代替ルート・再エネの現状
   - 政府の動き（最新）

5. 投資家が今すべきこと（2分）
   - 注目セクター・銘柄名（具体的に）
   - ヘッジ戦略（原油ETF、ゴールド、円安対応）

6. まとめ・クロージング（30秒）
   - 視聴者への行動喚起（チャンネル登録・コメント）

【出力形式】
- ずんだもんのセリフは「ずんだもん：〜なのだ！」形式
- めたんのセリフは「めたん：〜だよね」形式
- 各セクションの冒頭に【タイトル：〇〇】を付ける
- 重要な数値は**太字**にする
- テロップ用キーワードを各セクションに3〜5個リストアップする

ソースの情報を最大限活用し、数値の出典も括弧書きで明記してください。
```
