## 提案フォーマット

**5〜7件**のトピックを、以下のフォーマットで番号付きリストにして提案してください。

```
【番号】タイトル案（30字以内）

- テーマ: （1〜2行でトピックの核心を説明）
- なぜ今ホットか: （直近1〜2週間の動向・ニュースとの関連を説明）
- 視聴者へのバリュー: （投資家・ビジネスパーソンにとって何が役立つか）
- 想定タイトル案: （視聴者の興味を引くYouTubeタイトル案を1〜3本）
```

---

## 留意点

- 直近1〜2週間で実際に動きがあったニュース・発表に基づいて提案してください
- 日本の投資家視点で「自分事」として捉えられるトピックを優先してください
- 同じ領域のトピックが重複しないよう、バランスよく選んでください
