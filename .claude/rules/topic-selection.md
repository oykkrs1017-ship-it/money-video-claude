# トピック選定フロー（標準ルール）

## 原則

テーマを先に決めない。**直近1〜2週間の時事ニュースを起点にしてから**、地政学・投資角度を逆引きする。

地政学テーマ単独（「台湾海峡」「マラッカ海峡」など）は日本の一般投資家に刺さらない。
「今週のあのニュース → 自分のお金にこう影響する」という構造でないとクリックされない。

## 必須フロー

```
1. /research-topic を実行
   └─ 直近の政治・金融・AI系ニュースをスキャン
   └─ 競合チャンネルの急上昇動画を確認

2. 提案トピック（5〜7件）をユーザーに提示
   └─ 各トピックに「時事フック」「視聴者感情」を明示

3. ユーザーが番号を選択 → 台本生成フローへ
```

## トピックの評価基準

| 基準 | 良い例 | NG例 |
|------|--------|------|
| 時事フック | 「160円突破・為替介入発動」 | 「円安について解説」 |
| タイトル感情 | 損失回避 / FOMO / 逆説 | 教科書的な説明型 |
| 旬の有無 | イベント前後1週間以内 | 特定日付なし |
| 投資との接続 | 「この銘柄が動く」まで落とし込む | 地政学解説で終わる |

## スキャン対象（topic-researcher エージェントが自動収集）

- 政治：日米・米中・中東・NATO関連の直近動向
- 金融：為替・日銀・Fed・決算・指数急変
- AI・テック：大型投資発表・規制・半導体サプライチェーン
- 競合チャンネル：高橋ダン・両学長・テスタ等の直近急上昇動画

## 発動タイミング

- 次の ep を何にするか迷ったとき → 必ず `/research-topic` から始める
- ユーザーから「テーマ選びたい」「次どうする」と言われたとき
- 前回 ep のアップロード完了直後

## Why型タイトルへの変換（必須）

トピック選定後、タイトル候補を必ず **Why型 + 学術タグ複合** に変換する。

### フォーマット
```
なぜ{現象/銘柄/出来事}は{逆説的な結果}なのか【{学術分野1}×{学術分野2}】
```

### 学術タグ語彙集（組み合わせて使う）
- 地政学 / 行動経済学 / ゲーム理論 / 進化心理学 / マクロ経済学 / サプライチェーン工学

### 変換例
```
Before: 「東京エレクトロンが中国に売れない理由」
After:  「なぜ半導体規制は日本株投資家だけが損をする構造なのか【地政学×行動経済学】」

Before: 「ホルムズ海峡封鎖と日本のエネルギー安保」
After:  「なぜ日本だけが石油危機で真っ先に資産を失うのか【地政学×マクロ経済学】」
```

### title-optimizer agent との連携
トピック提案後に `title-optimizer` agent を呼び出してWhy型3案を生成する（Step 1.5）。
詳細: `.claude/rules/title-conventions.md`
