# トピック選定フロー（標準ルール）

## 原則

テーマを先に決めない。**直近1〜2週間の時事ニュースを起点にしてから**、地政学・投資角度を逆引きする。地政学テーマ単独は日本の一般投資家に刺さらない。「今週のあのニュース → 自分のお金への影響」という構造が必要。

**【王道KW着地ルール 2026-05-29】** 時事フック起点は維持しつつ、**最終的に王道KW（新NISA / iDeCo / S&P500 / オルカン / SBI証券 / 楽天証券 等）の検索対象へ着地させる**こと。地政学/時事＝アングル、王道KW＝検索主語。例「ホルムズ海峡封鎖 → 新NISAのS&P500はどうなる」。

**ターゲット層（2026-05-29拡大）**: 「30代個別銘柄投資家」→ **30〜60代の個人投資家、特に老後資金に不安を持つ会社員層**。老後・会社員・50-60代の「お金の不安」を自分ごと化できるテーマを優先する（ガーコちゃんねる分析）。

## 必須フロー

```
1. /research-topic を実行（直近ニュース + 競合急上昇動画をスキャン）
2. 提案トピック（5〜7件）を時事フック・視聴者感情付きで提示
3. ユーザーが番号を選択 → 台本生成フローへ
```

## evergreen 定番シリーズ lane（投稿の谷間を埋める安定供給 — ガーコちゃんねる分析 2026-05-28）

旬ニュースが弱い週は、時事起点に固執せず **evergreen 定番シリーズから1本選ぶ**。ニュース非依存で安定供給でき、新作の初速の母数を確保できる。ガーコちゃんねるの最大ヒットも定番evergreen（確定申告・年末調整＝毎年来る・損回避・全員対象）である。

| シリーズ | 性質 | 損回避タイトル型 |
|---------|------|----------------|
| 日本にしかない | 企業解剖（ニュース非依存） | 「〇〇を知らない投資家は損してる」 |
| チョークポイント | 地政学要衝 | 「〇〇封鎖！あなたの資産はどうなる？」 |
| 逆風で笑う企業 | 構造分析 | 「〇〇なのに〇〇だけ上がる理由」 |
| 資産防衛 | 為替介入/利上げ/選挙の度に再利用 | 「〇〇、NISAの米国株は今すぐ見直すべき」 |
| 制度活用（王道KW・2026-05-29追加） | 新NISA/iDeCo/SBI証券・楽天証券の改正・改悪 | 「〇〇の改悪、今すぐ確認」 |
| インデックス比較（王道KW・2026-05-29追加） | S&P500 vs オルカン vs FANG+ の比較 | 「{A} vs {B}、{数字}の差」 |

詳細定義: `knowledge/directive.yaml` の `evergreen_series`。

## トピックの評価基準

| 基準 | 良い例 | NG例 |
|------|--------|------|
| 時事フック | 「160円突破・為替介入発動」 | 「円安について解説」 |
| 旬の有無 | イベント前後1週間以内 | 特定日付なし |
| 投資との接続 | 「この銘柄が動く」まで落とし込む | 地政学解説で終わる |
| 王道KW流入 | 新NISA/S&P500/オルカン等を主語化 | 専門用語・ニッチ銘柄のみ |
| 期限・最新性 | 制度改正の締切前・「今すぐ確認」 | 日付なし・いつでも見られる |

## スキャン対象
- 政治：日米・米中・中東・NATO、金融：為替・日銀・Fed・決算、AI・テック：半導体・規制
- 競合：高橋ダン・両学長・テスタ等の直近急上昇動画
