# タイトル生成規則（ガーコちゃんねる5戦略を反映・2026-05-29全面改訂）

> SoT: `knowledge/directive.yaml` → `tech_geopolitics.script_rules.title`。型・王道KW変更は directive.yaml を編集。

競合「ガーコちゃんねる」（46万登録・直近平均18万再生）の詳細分析で判明した高CTR5戦略を反映する。**短く断定的な損回避型タイトル**を基本に、**王道キーワード（新NISA等）を主語**にし、**老後資金に不安を持つ会社員・50-60代を含む個人投資家**へ「自分の損得」を一瞬で伝える。

> **統合方針**: 地政学/時事＝アングル（フック）、王道KW＝検索対象＋自分ごとの着地点。
> 例「ホルムズ海峡封鎖、新NISAのS&P500はどうなる」「台湾有事、iDeCoの米国株は今すぐ見直すべき」。
> 地政学アングルを保ったまま、検索される主語を王道KWに置く。

## 優先順位

| 優先度 | パターン | テンプレート | 例 |
|--------|---------|------------|----|
| **1（最優先）** | 損回避・警告型 | `{ターゲット/王道KW}、{警告/やめるべき/大損}` | 50代60代、新NISAの米国株は今すぐ見直すべき |
| 2 | A vs B 比較対立型 | `{A} vs {B}、{数字}の差！` | S&P500 vs オルカン、10年で300万円の差 |
| 3 | 期限・最新性型 | `{2026年最新版/〇月まで}{王道KW}{警告}` | 2026年最新版！SBI証券の改悪、今すぐ確認 |
| 4 | ターゲット明示＋数字差型 | `{年代/属性}必見！{出来事}で{数字}の差` | 会社員必見！iDeCo改正で500万円の差 |
| 5 | 断定結論型 | `{王道KW/銘柄}、{断定的な好悪の結論}！` | オルカン、超進化します。／東京エレクトロン、黄金期突入！ |

タイトル候補は3案生成し、**最低1案は優先度1（損回避・警告型）／最低1案に王道KWを含める**こと。

## 語彙集（3カテゴリ）

### 損回避・断定ワード（優先度1・5で必ず1つ以上使う）
やめておくべき / 大損 / 大改悪 / 改悪 / 危険 / 今すぐ見直すべき / 知らないと損 / 手遅れになる / ここで買うと / 〜できなくなる / 罠 / オワコン / 悲劇 / 強制解除 / 貧乏 / 地獄 / 落とし穴 / 黄金期 / 超進化 / 〜の時代が来た / 突入

### 期待感ワード（ポジ・hookのH-00損失回避KW裏返しでも使用）
絶好の仕込み時 / 最高値更新 / 最強 / バーゲン / 爆益 / 黄金期 / 超進化 / 〜の時代が来た

### 期限・最新性ワード（優先度3で必ず1つ以上使う）
2026年最新版 / 〇月まで / 今すぐ / 絶対確認 / 今のうちに / 〜の前に / 手遅れになる前に

### ターゲット明示ワード（優先度1・4で活用）
50代・60代 / 会社員 / 老後 / 65歳までに / 定年前 / サラリーマン

## 共通ルール

- **30字以内**（サムネ表示幅）
- **先頭20字以内**に検索キーワード（**王道KW** または 銘柄名・地名・政策名）
- **【H-04】王道KW または 個別銘柄名を先頭20字に1つ以上必須**
  - 王道KW語彙: 新NISA / iDeCo / S&P500 / オルカン / FANG+ / SCHD / 高配当ETF / SBI証券 / 楽天証券 / つみたて投資枠 / 成長投資枠
  - 個別銘柄: TSMC / NVIDIA / ASML / 東京エレクトロン / 信越化学 / INPEX / 三菱重工 / ファナック / ディスコ / レーザーテック 等（地政学アングル回では銘柄名を併用し差別化を残す）
- **断定・感嘆符「！」・損回避ワード**で引きを作る（疑問符より断定を優先）
- 生成した各案について、王道KWまたは銘柄名が先頭20字以内にあるか自己チェックし、無い案は破棄して再生成する

## 廃止（2026-05-28）

「なぜ〇〇は〇〇なのか【学術1×学術2】」型（Why型+学術タグ複合）の最優先指定は廃止。学術タグ語彙集も使用しない。Why型は「知的だが損得が一瞬で伝わらない」ためCTRで劣ると判断（ガーコ実データ）。
