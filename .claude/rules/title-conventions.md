# タイトル生成規則

## 優先順位

| 優先度 | パターン | テンプレート | 備考 |
|--------|---------|------------|------|
| **1（最優先）** | Why型 + 学術タグ複合 | `なぜ{現象}は{逆説的な結果}なのか【{学術1}×{学術2}】` | 考えすぎる葦分析で333k再生を確認 |
| 2 | 逆説型 | `{銘柄}が{衝撃の事態}→なのに{逆の現象}が起きる理由` | 最高再生数実績あり（既存） |
| 3 | 数字インパクト型 | `{X}兆ドル{現象}！{銘柄/地名}が{変化}する{期間}` | SEO重視時に選択 |
| 4（補助） | 隠された真実型 | `{X}%が知らない{現象}の正体` | — |
| 4（補助） | 唯一性型 | `{X}が変わったとき、唯一{メリット}を得る{条件}` | — |
| 4（補助） | 年予測型 | `{年}年{予測}：{銘柄}に何が起きるか` | — |

## Why型のルール

### フォーマット
```
なぜ{A}は{B}なのか【{学術分野1}×{学術分野2}】
```

- **A**: 具体的な現象・企業・人物・政策（「日本の電気代」「半導体規制」「円安」）
- **B**: 逆説的・衝撃的な結果（「日本人だけが損をする」「資産が10年で半減する」「最強の投資機会になる」）
- **学術タグ**: 2分野を組み合わせる。視聴者が「ちゃんとした根拠がある」と感じる信頼感を演出

### 学術タグ語彙集

| タグ | 使用場面 |
|------|---------|
| 地政学 | 国家間紛争・規制・外交 |
| 行動経済学 | 投資家心理・バイアス・市場の非合理性 |
| ゲーム理論 | 貿易戦争・囚人のジレンマ・交渉 |
| マクロ経済学 | 為替・インフレ・金融政策 |
| サプライチェーン工学 | 半導体・部品不足・物流 |
| 進化心理学 | 人間行動・集団心理・リスク認知 |

## タイトル生成フロー

```
topic-researcher → title-optimizer agent → 3案提示
                                          ├─ Why型（最優先）
                                          ├─ 逆説型（バックアップ）
                                          └─ 数字インパクト型（SEO重視）
```

## 共通ルール（全パターン）

- **30字以内**を目安（サムネ表示幅）
- **先頭20字以内**に検索キーワード（銘柄名・地名・政策名）
- **タイトルに銘柄名必須**（INPEX・三菱重工・東京エレクトロン等）→ SEO + 投資意識層にリーチ
- 疑問符「？」か「のか」で終わる形式を優先（クリック欲求を残す）
