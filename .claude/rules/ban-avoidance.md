# 動画品質ルール（YouTube BAN回避）

- VariationEngineを必ず通す。seed値が異なれば全て異なる見た目
- 静止画スライドショー禁止。全要素に最低1つのアニメーション
- キャラクターは常にアイドルアニメーション（呼吸）
- チャート・データはナレーションと同期アニメーション

## 視聴者体験ルール（AI生成感を出さない）

- **チャプタータイプ名の表示禁止**: "hook", "cta", "explanation" 等の制作構造ラベルを画面に出さない（ChapterCard は削除済み）
- **CTA は最終チャプターのみ**: 中盤への「チャンネル登録」挿入は視聴体験を壊す。`chapters` の末尾にのみ配置する
- **許可される UI**: TopicBadge（トピック名のみ）/ ProgressBar / SubtitleLayer

# YouTube Inauthentic Content Policy 対策（2026-05-31 昇格）

## 規制の要点

2026年1月、YouTubeは「Inauthentic Content Policy」（2025年7月に旧・繰り返しコンテンツ規約を改名・拡大）で**史上最大のAIチャンネル一斉収益化停止**を実行。

- **連鎖停止ルール（2025-10施行）**: 1チャンネルの違反が同一AdSense配下の**全チャンネルBAN**に波及する。原因チャンネルの削除は救済手段を失うため絶対に避ける。
- **AI利用自体は禁止ではない**。停止対象は「独自性・付加価値のない量産」。

## 本チャンネルが該当する危険シグナル5項目（全該当＝最高リスク帯）

| シグナル | 該当 | 対策の柱 |
|---|---|---|
| 紙芝居型スライド解説 | HTMLスライド | スライド脱テンプレ化（柱B） |
| TTS単独ナレーション | VOICEVOX | 独自性で付加価値立証（方針: 維持） |
| 固定立ち絵・背景使い回し | ponchan/maro固定キャラ | 表示頻度・バリエーション |
| 同一テンプレ量産 | prompt.ts定型台本 | テンプレ脱却（柱A） |
| ネット要約の焼き直し | リサーチ→自動台本 | 独自視点・独自試算（柱A） |

## 投稿前チェックリスト（毎ep必須）

- [ ] **独自視点**（"私はこう読む"）が analysis に最低1つあるか
- [ ] **独自試算**（一次データに基づく自前計算・算出過程つき）が最低1つあるか
- [ ] **想定反論への応答**が1箇所あるか
- [ ] 直近epと**言い回し・数字提示フォーマット・hook H-05型**が被っていないか
- [ ] スライドの**テーマ／レイアウト**が前epと変わっているか
- [ ] 説明文に**AI開示ラベル・制作意図・教育的価値**が入っているか

> SoT: `knowledge/directive.yaml` → `script_rules.originality`。変更は directive.yaml を編集。

## 投稿頻度ルール

**週2〜5本・量より独自性**。毎日テンプレ投稿は「volume over substance」シグナル。
1本ごとに編集的バリエーション（独自考察・構成・見た目）を確保すること。尺は [[video-length]]（8〜11分）と併記。

## 危険パターン禁止リスト

- ネット要約のみで独自考察ゼロの台本
- 同一の数字提示フォーマット（「○兆ドル——」型）の連投
- 毎日・同一テンプレの大量投稿
- 固定立ち絵・固定背景・固定配色を全epで完全に同一にすること
