# Hookチャプター構造規則

## 3段構成（30秒Hook内の時間分割）

| 時間帯 | ルール番号 | 名称 | 内容 | 担当キャラ |
|--------|---------|------|------|-----------|
| **0〜15秒** | **H-05（新規）** | 逆説層 | 逆説的な問い OR 信じられない数値を即投下 | ponchan |
| **15〜25秒** | H-01 | 結論層 | 結論 + 衝撃数字 | ponchan |
| **25〜30秒** | H-03 | 学べる3つ層 | rich-panelで「この動画で学べる3つのこと」を提示 | maro |

## H-05: 逆説層のルール（新規追加）

### 目的
考えすぎる葦チャンネル分析（333k再生）を参考に、冒頭15秒で視聴者の「え、なんで？」を引き出す。

### セリフ生成パターン

**パターンA: 逆説的な問い型**
```
「もし{前提条件}が{変化}したら、あなたの{資産/生活}はどうなると思う？」
例: 「もし明日、日本の石油輸入が全部止まったら、スーパーの値段はどうなると思う？」
```

**パターンB: 信じられない数値型**
```
「実は{対象}の{数値/パーセンテージ}が、もう{衝撃的な状態}なんだよ」
例: 「実は日本が輸入する石油の95%が、たった一つの海峡を通ってるんだよ」
```

**パターンC: 常識覆し型**
```
「{一般的に信じられていること}って思ってる人が多いけど、実はその逆なんだよね」
例: 「半導体規制って中国を困らせるためのものって思ってるでしょ？実はその逆で、日本株投資家が一番損してるんだよね」
```

### 2ライン構成（0〜15秒）

```
Line 1（0〜7秒）: パターンA/C の逆説的な問い（ponchan）
Line 2（7〜15秒）: パターンBの信じられない数値で補強（ponchan）
```

## 既存ルール（継続）

### H-01: 結論+衝撃数字（15〜25秒）
- ponchan 1発目で「結論 + 衝撃数字」を明示
- 例: 「今日のポイント：〇〇で日本株が最大XX%動く可能性があるよ」

### H-03: rich-panel（25〜30秒）
- maro が「この動画で学べる3つのこと」を提示
- rich-panel ビジュアルと同期

### H-02: 尺規則
- Hook チャプター全体: 30秒固定（ep004〜継続）

### H-04: タイトル銘柄名必須
- Hook のセリフに動画タイトルに入れた銘柄名を必ず1回以上言及

## Remotion連携

Hook チャプターの最初2ライン（0〜15秒）は `CounterIntuitionLayer` が優先表示される。
詳細: `packages/tech-geopolitics-channel/src/components/CounterIntuitionLayer.tsx`
