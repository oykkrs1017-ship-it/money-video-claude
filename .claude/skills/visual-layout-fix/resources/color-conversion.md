# ライト化カラーリファレンス

## 変換テーブル

| 用途 | 旧（ダーク） | 新（ライト） |
|------|------------|------------|
| 主要テキスト | `#ffffff` | `#1a1a3e` |
| サブテキスト | `#aaaaaa`, `#cccccc` | `#444466` |
| 箇条書きテキスト | `#cccccc` | `#333355` |
| ボックス背景 | `rgba(0,0,0,0.xx)` | `rgba(255,255,255,0.92)` |
| フッター背景 | `rgba(255,255,255,0.06)` | `rgba(30,40,100,0.10)` |
| フッターテキスト | `#cccccc` | `#2a2a4a` |
| SVG 構造線 | `rgba(255,255,255,0.xx)` | `rgba(40,60,120,0.xx)` |
| SVG メインライン | `#ffffff` | `#1a1a3e` |

## 症状→修正対象 早見表

| 症状 | 原因 | 修正 |
|------|------|------|
| テキストが白/グレーで見えない | ダークテーマ残留 | 全テキストカラーを `#1a1a3e` 系に変更 |
| 背景が黒/暗色 | `rgba(0,0,0,...)` 残留 | `rgba(255,255,255,0.92)` に変更 |
| テキストが細い | `fontWeight` 未指定 | 全要素に `fontWeight: 'bold'` |
| グリッドセルが上寄り | CSS Grid デフォルト | `display:flex, alignItems:center` 追加 |
| ノードが枠外はみ出し | 固定幅計算オーバーフロー | `minWidth`→小さく or `maxStep`→小さく |
