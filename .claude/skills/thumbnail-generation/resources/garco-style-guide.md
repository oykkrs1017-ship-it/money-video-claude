# ガーコスタイル設計標準（正式標準 2026-05-30採用）

> 根拠: ガーコちゃんねる（46万登録・直近平均18万再生）の分析から、文字が70-80%を占めるスタイルが最高CTRを達成（ep021サムネイル制作時に正式採用）。

## デザイン原則

| 要素 | 仕様 |
|------|------|
| 文字占有率 | 70-80%（背景は文字の「台座」） |
| 背景 | 単色 or 極めてシンプルなグラデーション（濃紺`#0A0F2E` or 黒`#080808`） |
| テキスト行数 | 4行スタック + 1帯（計5要素） |
| フォント | Noto Sans JP Black（最太字）相当、アウトラインのみ（シャドウ禁止） |
| 配色 | 白`#FFFFFF` + 黄`#FFE500` + 赤強調/緑帯 |

## パターン種別

| パターン | 用途 | 帯色 |
|---------|------|------|
| **案1: 損回避・警告型** | NISAや積立の見直し、改悪、注意喚起 | 赤`#CC0000` |
| **案2: VS比較対立型** | ファンド比較、商品比較、A vs B | 緑`#16a34a` |

## 案1（損回避・警告型）テキスト構成

```
L1: {王道KW}の              白・60-70px
L2: {銘柄/KW本体}           黄#FFE500・110-130px
L3: 今すぐ                  白・最大サイズ
L4: {警告ワード}             白・130-150px
帯: {数字}の衝撃             白文字・赤帯#CC0000
```

## 案2（VS比較対立型）テキスト構成

```
L1: {左ターゲット}           黄#FFE500・90-110px
L2: VS                      赤#FF3333・180-200px（最大・支配的）
L3: {右ターゲット}           白・100-120px
L4: {年号}年                白・65-75px
帯: {結論ワード}！           白文字・緑帯#16a34a
```
