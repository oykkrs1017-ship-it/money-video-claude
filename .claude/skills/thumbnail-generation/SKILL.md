# スキル: サムネイル生成プロンプト設計
最終更新: 2026-04-20（ep007で超高密度フォーマットに更新）

## 最優先原則（不変）
1. **キャラクター絶対禁止** — まろくん・ぽんちゃんはサムネイルに入れない。No anime, no characters, ever.
2. **著名人・シルエット禁止（Google Flow ポリシー）** — 人物名（日本語・英語問わず）をプロンプトに含めると Google のポリシー違反で生成失敗する。「マスク」「Elon Musk」「安倍」等の固有名詞はプロンプトから完全除去。シルエットも「人物描写」とみなされリジェクトされるリスクあり。→ 人物なし・顔なし・シルエットなしを徹底する。
3. **超高密度プロンプト必須** — 短いプロンプト禁止。セクション分割・仕様記述・テキスト指定を全て含めた構造化プロンプトを出力する
4. **テキスト埋め込み** — Google Flow はテキスト生成精度が高い。日本語テキストをプロンプトに直接記述してサムネイルに焼き込む
5. **日本語テキスト強制** — プロンプト冒頭に必ず「出力する画像内テキストは必ず日本語で出力して」を追加する。これがないとAIが英語テキストを生成する（ep014で確認済み）。
6. **CTR最大化** — 「見た瞬間に止まる」「見ないと損」を設計する

## 超高密度プロンプトの構造（必須フォーマット）

サムネイルプロンプトは必ず以下の**7セクション**を含む構造化テキストで出力する。

**プロンプト冒頭に必ず追加する1行（日本語テキスト強制）:**
```
出力する画像内テキストは必ず日本語で出力して
```

```
=== BACKGROUND VISUAL ===
[背景・メインビジュアルの詳細記述]
[色パレット: HEXコード指定]
[照明・雰囲気・スタイル]

=== TEXT OVERLAYS (Japanese) ===
[配置場所 — テキスト内容]
[配置場所 — テキスト内容]
...

=== TYPOGRAPHY SPECS ===
[フォント太さ・サイズ感・stroke・shadow の詳細]
[コントラスト要件]
[320px幅での可読性要件]

=== COMPOSITION ===
[三分割法での配置指定]
[テキスト配置ゾーン]
[奥行き・視線誘導の指定]

=== EMOTIONAL TRIGGERS ===
[狙う感情: 恐怖/緊急/好奇心/損失回避]
[視聴者の心理状態の目標]

=== STYLE REFERENCE ===
[参照スタイル（Bloomberg/Wired/NHK等）]
[No characters指定]
[解像度・クオリティ指定]

=== AVOID ===
[除外要素リスト]
```

## テキストオーバーレイの設計原則

### 配置ゾーン（16:9）
| ゾーン | 用途 | スタイル |
|--------|------|----------|
| 上部1/3 | メインタイトル（大） | 白文字＋黒縁＋赤シャドウ |
| 下部左 | サブキーワード / 数字バッジ | 黄色角丸バッジ |
| 下部右 | 緊急スタンプ | 赤バッジ / 警告マーク |
| 中央帯 | 補足ラベル | 半透明黒帯＋白文字 |

### CTR最大化テキスト設計（優先順位付き・2026-05-17更新）
| 優先度 | パターン | 例 | 心理効果 |
|--------|----------|-----|---------|
| **1（最優先）** | **「なぜ〜なのか」+ 学術タグ** | 「なぜ日本株だけ損するのか」＋小文字で「地政学×行動経済学」 | 好奇心ギャップ＋権威性 |
| 2 | 逆説型 | 「規制されたのに最高益」 | 常識崩壊＋好奇心 |
| 3 | 数字＋衝撃 | 「95%依存の真実」 | 具体性＋恐怖 |
| 補助 | 「〜する日」 | 「暗号が破られる日」 | 具体性＋期限感 |
| 補助 | 「〜限界説」 | 「2030年限界説」 | 数字＋権威性 |
| 補助 | 「〜崩壊シナリオ」 | 「金融崩壊シナリオ」 | 損失回避バイアス |

### Why型サムネパターン（2026-05-17追加・最優先）

考えすぎる葦チャンネル（333k再生）の分析から、Why型サムネが最高CTRを達成することを確認。

**テキストレイアウト:**
```
上部: 「なぜ○○は」（大字・白＋黒縁）
中部: 「○○なのか？」（最大字・アクセントカラー）
下部小: 「地政学 × 行動経済学」（バッジ風・小字・金色）
```

**配色原則（Why型専用）:**
- 背景: 深い紺か黒（「権威感」「知的好奇心」を喚起）
- メインテキスト: 白または金（可読性最大）
- 学術タグバッジ: 金色または琥珀（「ちゃんとした根拠感」）
- アクセント: 赤かオレンジ（疑問符・ポイント強調）

**サンプルプロンプト（Why型）:**
```
=== TEXT OVERLAYS (Japanese) ===
上部中央 — 「なぜ半導体規制は」（白・太字・56px相当）
中央大 — 「日本株だけ損するのか？」（金色・極太・80px相当・黒縁4px）
下部バッジ — 「地政学 × 行動経済学」（小字・金背景バッジ・角丸）

=== EMOTIONAL TRIGGERS ===
好奇心ギャップ（「自分だけ損しているかも」という焦り）
自己照合（「これ俺の話だ」と感じさせる）
権威性（学術タグで「ちゃんとした根拠がある」印象）
```
