# 学び: フィードバックの適用範囲を局所化しない
日付: 2026-05-04
カテゴリ: 上位原則候補
関連スキル: 全スキル横断

## 状況
ユーザーが「2196フレームあたりのチャンネル登録促進はこのタイミングでは不要」と指摘。
Claude は「ep013 のこの箇所では不要」と解釈して削除したが、
ユーザーの真意は「途中のCTAは常に不要、最後のみOK」という恒久ルールだった。

## 齟齬の内容
- AI の判断: 「このタイミングでは」＝特定フレーム・特定エピソードの局所的修正
- ユーザーの意図: 「このタイミングでは」＝中盤挿入というパターン自体を否定する恒久ルール
- 差分の本質: ユーザーが個別事例を指摘するとき、その背後に「パターン全体を変えてほしい」意図がある場合が多い

## 学び
ユーザーからの修正指示を受けたとき、「この1箇所だけ直す」で終わらず
**「これはパターン全体のルール変更か？」を必ず自問する**。
特に「このタイミングでは」「今回は」という言葉があっても、
同じ問題が構造的に発生するなら恒久ルールとして扱う。

判断フロー:
1. 修正指示を受ける
2. 「同じパターンが他の箇所・他のepでも発生しうるか？」を確認
3. Yes → 恒久ルールとしてメモリ・lessons.md に記録
4. No → 局所修正で完結
