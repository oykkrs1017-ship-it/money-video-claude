# 学び: 台本生成後の4項目品質チェックリスト
日付: 2026-04-26
カテゴリ: 個別の学び
関連スキル: research-topic, generate-script

## 状況
ep009の1回目台本（東京エレクトロン解説型）を生成・プレビューした後、
ユーザーから「解説型で人は動かない。視聴者の自分事化が足りない」と指摘。
台本を丸ごと再生成することになった（コスト・時間の無駄）。

## 齟齬の内容
- AI の判断: 「良い題材 + 正確な情報 = 良い台本」
- ユーザーの意図: 「視聴者が恐怖を感じ、今すぐ行動したくなる台本」
- 差分の本質: YouTubeはエンタメ。情報の正確さより感情駆動（恐怖・期待・自己利害）が視聴維持率を決める

## 学び: 台本生成直後に必ずこの4項目でセルフスコアリングする

### チェックリスト
| 項目 | チェック観点 | NGの兆候 |
|------|-------------|---------|
| ① 恐怖ワード | タイトル・フックに「知らない」「消える」「損する」「差がつく」等があるか | 「〜について解説します」で始まる |
| ② 特定数字＋ペルソナ | 「○○%の△△民」「▲▲円持っている人」のように数字とターゲットを組み合わせているか | 「多くの人が」「一般的に」の曖昧表現 |
| ③ 行動出口 | 動画を見終わった視聴者が「今日できること」を理解できるか | まとめが「〜が重要です」で終わる |
| ④ 情報格差の演出 | 「なぜ誰も言わないのか」「機関投資家は知っているが個人は知らない」という非対称性があるか | 教科書・Wikipedia的な中立的解説 |

### スコアリング基準
- 4項目全部OK → 生成物として合格、次工程へ
- 2〜3項目OK → 修正箇所を特定してセリフ単位で修正
- 0〜1項目OK → 台本の角度を変えて再生成（コストより品質優先）
